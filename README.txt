Setup and Execution Instructions
1. Download 'index.html', 'script.js' and 'style.css'
2. Open 'index.html' file
3. Enter common color name (e.g., "red", "green") on the input field 
4. Click the "Enter" button
5. The hexadecimal code will be displayed. If not, an alert will be shown.

Design Decisions 
1. User Interface: A clean and simple format that is easy for users to navigate and use the tool
2. Include CSS and JavaScript for a cleaner code

Technologies Employed
HTML, CSS, JavaScript

Testing Execution
1. clone the repository
2. ensure npm and jest are installed by keying "npm install --save-dev jest" on the terminal window
3. run the test by keying "npm test" on the terminal window
